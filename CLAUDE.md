# CLAUDE.md

Guidelines for Claude Code working in this repository.

## Communication

- Respond in Japanese
- Be concise and direct

## Coding

- Prefer simplicity over excessive robustness
- Follow existing code style and patterns

## Commits & PRs

- PR descriptions should be concise, clear, and in Japanese
- Pass lint/format checks before pushing

## Structure

- `nvim/` — Neovim configuration
- `wezterm/` — WezTerm terminal configuration
- `hammerspoon/` — Hammerspoon window management
- `claude/` — Claude Code global settings (symlinked to `~/.claude/`)
- `.zshrc` — Zsh configuration
- `.tmux.conf` — tmux configuration
- `starship.toml` — Starship prompt configuration
- `set-symboliclink.sh` — Symlink setup script
